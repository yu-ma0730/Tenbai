MT5 リアルデータ接続ガイド
  1. Windowsマシンに MetaTrader5 をインストール・ログイン
  2. Python をインストール: pip install MetaTrader5 requests
  3. このリポジトリの mt5_bridge.py を Windows にコピー
  4. 実行: python mt5_bridge.py --server http://このサーバーのURL:5000 --interval 15
  5. 接続すると上部バーが MT5 接続中 に変わります
ブローカーによってシンボル名が異なる場合（例: XAUUSD → GOLDm）は --symbols USDJPY EURUSD EURJPY GOLDm で指定可能です。
⚠ 免責事項: 本ツールはローソク足パターンを自動検出してサインを表示しますが、FX取引には相場リスクが伴います。 本ツールの情報は投資助言ではなく参考情報です。実際の取引はご自身の判断と責任でお行いください。 現在のデータはデモ用シミュレーションデータです。実運用にはブローカーAPIへの接続が必要です。

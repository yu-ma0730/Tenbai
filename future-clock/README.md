# 未来時間時計 (Future Clock)

Androidアプリ — 欠勤管理タイマー

## 機能

- **リアルタイム時計** — 日付・時刻を1秒ごと更新
- **欠勤予定日の選択** — カレンダーから選択（本日以降のみ）
- **カウントダウン** — 日・時・分・秒で残り時間を表示
- **欠勤種別の自動判定**
  - X日以上前 → ✓ 欠勤登録（緑）
  - X日未満   → ⚠ 当日欠勤（赤）
  - 当日       → ！当日欠勤（オレンジ）
- **X日前の閾値を自由入力**（デフォルト: 10日）
- **アラート通知** — 指定した日の午前9時に通知
- **端末再起動後もアラーム復元**

## セットアップ

1. Android Studio でこのフォルダ（`future-clock/`）をルートとして開く
2. Gradle sync を実行
3. 実機またはエミュレータで Run

## 動作要件

- minSdk: 26 (Android 8.0)
- targetSdk: 34 (Android 14)
- Kotlin 1.9
